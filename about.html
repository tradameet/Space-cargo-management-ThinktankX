ThinkTankX Universe is a futuristic Space Station Cargo Management System designed to handle essential resources, personal needs, and mission-critical equipment with precision and efficiency.

This project includes four major modules: Consumables, Equipment, Payload, and Personal & Psychological Inventories. Each module is represented with an interactive web interface that allows astronauts and mission control to monitor and manage resources in real time.

Built using HTML, CSS, JavaScript, and node.js, the system ensures clean visuals, responsive design, and data visualization through dynamic tables, search functionality, and pie charts.

üéØ Project Goals

The core objective is to simulate how inventory is tracked and managed aboard a space station. The system focuses on:

  • Intuitive user interface (UI)
  • Real-time item lookup
  • Graphical representation of remaining quantities
  • Supporting decision-making and mission success
